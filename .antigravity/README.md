# Antigravity Instructions & Context

This directory contains instructions and persistent context for the Antigravity AI agent.

- **instructions/**: Core behavioral guidelines and project-specific workflows.
- **knowledge/**: Placeholder for any repository-specific Knowledge Items (KIs) that should persist across sessions.

## Usage
Antigravity should read these files at the beginning of a task to understand the preferred coding style, communication patterns, and algorithmic constraints of this repository.
